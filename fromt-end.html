What is a Programming language?

First, let us start by definning programming:
Programming is a process by which a person gives the computer a certain task to
perform. But having an idea and not knowing how to
communicate it to the computer makes it all vague. This is where these languages come in: they
help us communicate with the computer.
But HTML and CSS are not programming languages,
they are scripting languages.

What then is a Framework?

Would you like to build a chair by going to the forest to fetch for woods or you would
prefer to have all the woods cut for you in their right sizes? That's where frameworks come in:
they contain in build libraries which facilitate the use of a programming language
