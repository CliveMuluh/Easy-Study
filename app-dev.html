What is a web application ?

A web application is a software which has a specific role it plays. I am sure by now you may have noticed that some languages can be used for many purposes. Alot of beginners get really excited and begin studying immediately. Well that fine, but it is always good to plan ahead of time as programming gets really tough with time.

how do I go about it?

Keep in mind that the technologies(languages) will always change so studying the language immediately is not always the best approach. The first thing to do is to have a project in mind. Studying a language with a project in mind sets on a particular and makes you focus. That way you ahev something to show especially when going for a job interview. Remember: no body cares to know how long you studied java but what did you used the java for.
